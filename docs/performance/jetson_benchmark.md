# Jetson TensorRT Benchmark

Date: 2026-06-01  
Device host: `nvidia-desktop`  
Jetson Linux: `R36.4.7`  
Kernel: `5.15.148-tegra`  
TensorRT: `10.3.0`  
CUDA toolkit: `12.6`  
Model: `outputs/models/unet_demo.onnx` exported from the local Tiny U-Net scaffold  
Input shape: `masked_image:1x1x128x128`  
Precision: FP16 engine build enabled with `--fp16`

## Command

```bash
/usr/src/tensorrt/bin/trtexec \
  --onnx=models/unet_demo.onnx \
  --saveEngine=models/unet_demo_fp16.engine \
  --fp16 \
  --shapes=masked_image:1x1x128x128 \
  --avgRuns=20 \
  --duration=10
```

## Notes

This benchmark uses random input generated by `trtexec`. It validates TensorRT compatibility and raw inference latency, not reconstruction quality. The next useful measurement is ONNX Runtime vs TensorRT on the same synthetic input, followed by a fastMRI sample once dataset transfer is available.
